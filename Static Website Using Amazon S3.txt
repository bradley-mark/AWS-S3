Amazon S3

>Create bucket
Bucket name: www.markbradley.cloud
AWS Region: US East (N. Virginia) US-east-1
Block Public Access settings for this bucket > Unselect - Block all public access - turns off block all public access
Leave all other defaults

> Objects
Upload > Add Files
error.html
index.html
Standard storage class

> Properties
Static website hosting > Enable
Host a static website
Enter index.html and error.html S3 files
Bucket website endpoint
http://www.markbradley.cloud.s3-website-us-east-1.amazonaws.com

> Permissions

Bucket policy > Edit

{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "PublicReadGetObject",
            "Effect": "Allow",
            "Principal": "*",
            "Action": "s3:GetObject",
            "Resource": "arn:aws:s3:::www.markbradley.cloud/*"
        }
    ]
}
